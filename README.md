# AliyunAccessKeyTools

Alibaba Cloud AccessKey Leak Exploitation Tool

This tool is used to query the host associated with the ALIYUN_ACCESSKEY and execute remote commands.

For information on how to use the ALIYUN_ACCESSKEY, please refer to the article: [Recording an Incident of Access Key Leakage Leading to Shell Access on an Alibaba Cloud Host](https://www.freebuf.com/articles/web/255717.html)

Support Status:

>- [x] Linux
>- [x] Windows
